# 药品记录管理系统 - 用户故事

## 功能列表

### 1. 药品拍照识别
```
As a 用户,
I want to 拍照识别药品名称,
so that 我可以快速记录药品信息。
```

### 2. 药品有效期管理
```
As a 用户,
I want to 记录和查看药品有效期,
so that 我知道哪些药品即将过期。
```

### 3. 吃药记录
```
As a 用户,
I want to 记录每次吃药的时间,
so that 我可以追踪服药历史。
```

### 4. 病例记录
```
As a 用户,
I want to 记录每次就诊的病例信息,
so that 我可以保存就医记录。
```

## 验收标准

- [ ] 可以拍照并自动识别药品名称
- [ ] 可以手动输入药品信息
- [ ] 可以设置药品有效期
- [ ] 可以记录每次吃药的时间
- [ ] 可以查看服药历史记录
- [ ] 可以记录病例信息
- [ ] 可以查看病例历史
- [ ] 数据本地保存
